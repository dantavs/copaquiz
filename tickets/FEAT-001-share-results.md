# FEAT-001: Expandir compartilhamento de resultados

## User Story
Como usuário, quero compartilhar meus palpites da fase eliminatória, para que meus amigos possam ver todo o caminho da competição que previ.

## Descrição
Atualmente, a funcionalidade de compartilhamento de resultados exibe apenas o campeão. A funcionalidade deve ser expandida para exibir todos os resultados preenchidos na fase eliminatória (oitavas, quartas, semifinal, final).

## Critérios de Aceitação
- [ ] O componente de compartilhamento deve listar os resultados de todas as rodadas da fase eliminatória.
- [ ] A imagem/texto gerado para compartilhamento deve estar legível e organizada.
- [ ] A funcionalidade deve continuar funcionando para o compartilhamento de apenas o campeão, caso o usuário não tenha completado a fase eliminatória.
