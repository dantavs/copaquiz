# UX-001: Melhorar transição entre fases do simulador

## User Story
Como usuário, quero um fluxo de transição entre as fases do simulador mais fluido, para que eu não precise realizar interações desnecessárias ou longas rolagens de tela.

## Descrição
A transição atual entre a fase de grupos e as fases eliminatórias exige que o usuário, após finalizar o preenchimento, role a página de volta para o topo para prosseguir. Isso gera atrito na experiência.

## Critérios de Aceitação
- [ ] Implementar uma melhoria que automatize a transição ou facilite o acesso para a próxima fase.
- [ ] A solução deve ser intuitiva e reduzir o esforço físico (rolagem) do usuário.
- [ ] Garantir que o estado do preenchimento da fase atual seja mantido durante a transição.
